# End-to-End-Hospital-Operations-Dashboard-and-SQL-Analysis
End-to-end healthcare analytics project that leverages Python, SQL, SQLite, and Streamlit to analyze hospital operations data, uncover trends in patient volume, wait times, treatment costs, and readmission rates, and deliver actionable insights through an interactive dashboard.


Performed categorical data cleaning to standardize inconsistent values across severity levels, departments, and admission types, eliminating duplicate filter categories caused by casing and whitespace inconsistencies.
